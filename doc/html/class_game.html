Detailed Description

Podstawowy interfejs do uruchomienia gry.

Gra odbywa się pomiędzy dwoma graczami przedstawianymi przez paletki W zależności od preferowanej metody wejścia, powinno się używać którejś z klas potomnych: KeyboardPlayedGame lub USBPlayedGame

Gra rozgrywana jest do momentu gdy któryś z graczy zdobędzie dwa razy sumę 11 punktów
