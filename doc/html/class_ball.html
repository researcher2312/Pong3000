Detailed Description

Klasa przedstawiająca piłkę

Piłka odbija się między graczami piłka sama porusza się w zadanym kierunku. Piłka przy dotknięciu paletki gracza odbija się i wydaje dźwięk, przy dotknięciu ściany górnej lub dolnej odbija sie ale nie wydaje dźwięku, przy dotknięciu ściany prawej lub lewej emituje sygnał ballOut
